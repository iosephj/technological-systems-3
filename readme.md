# Canvas LMS – Course Repository

Repository used to store and manage materials for a Canvas LMS course that supports face-to-face secondary-level classes.

## Structure and usage

- HTML files contain short reading guides and supporting materials.
- Content is designed to be lightweight and readable on mobile devices.
- HTML files are linked from Canvas pages, assignments, or modules.

## Interactive activities

- Interactive exercises are written in a plain text format.
- These text files are converted into QTI packages using **text2qti**.
- The resulting QTI files are imported into Canvas LMS as quizzes or assignments.

This workflow allows editing activities as text and re-importing them into Canvas when needed.

## Curriculum notes

- The course follows a spiral curriculum.
- Some lessons are still missing or under development and will be added progressively.
- Activities may be reused or adapted across modules as the course evolves.

## Language

- Course content is in Spanish.
- Notes and documentation may be written in English when convenient.
